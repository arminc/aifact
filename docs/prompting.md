# Prompting

The workflow intentionally stays on fixed OpenAI GPT model families for consistency.

Prompt behavior matters a lot, so stable model targets make prompt tuning easier and make workflow changes easier to reason about.

This scaffold includes two prompt reference files under `.opencode/custom/init/`:

- `openai-gpt-5-codex-guidlines.md`
- `openai-gpt-5-guidlines.md`

These files are prompt-tuning references for the agents in this workflow. They are not end-user product documentation.
